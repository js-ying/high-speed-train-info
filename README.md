# 高鐵時刻查詢
查詢高鐵時刻，簡單又快速。 

Demo: [https://hs-traintime.jsy.tw](https://hs-traintime.jsy.tw)

Ver.1 版本特色 (2021-11)
1. 快速查詢：打字或點擊即可篩選車站
2. 歷史查詢：最新六筆查詢紀錄
3. 票價查詢：包含團體與法優票價
4. 車次詳細時刻表：包含該車次起迄間所有進站時刻
5. 當日過期車次特效：已小於當下時間的車次會以半透明方式呈現
6. 這是我第一次挑戰深色佈景的設計，希望您會喜歡 😍
